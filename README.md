# myOrganizer-CURATOR

CURATOR is a PowerShell-based collection organization tool currently being prepared for public release.

The project is not available for download yet.

This repository is a public placeholder for release announcements, licensing information, documentation, and future source-code availability.

## Current Status

CURATOR is in private testing and documentation review.

## Planned License

CURATOR is expected to be released under the GNU General Public License v3.0 or later.

## Safety Notice

CURATOR reorganizes, stages, extracts, packages, and writes files.

When released, users should run CURATOR only against copies of their original files and should review all output before replacing anything.

## Availability

Source code, documentation, and release packages will be published when they are ready.
